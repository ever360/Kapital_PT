# Kapital App

Sistema de gestión para el control de rutas, personal y cuotas.

## Características Principales

- **Rediseño Premium**: Interfaz moderna con efectos de Glassmorphism.
- **Gestión de Equipo**: Control detallado de personal y asignación de roles.
- **Control de Cuotas**: Monitoreo en tiempo real del límite de usuarios activos.
- **Seguridad**: Autenticación biométrica e integración con Supabase.

## Comenzando

Esta aplicación está desarrollada con Flutter y utiliza Supabase como backend.

### Requisitos

- Flutter SDK (versión más reciente).
- Cuenta de Supabase configurada.

### Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/ever360/Kapital_PT.git
   ```
2. Instala las dependencias:
   ```bash
   flutter pub get
   ```
3. Ejecuta la aplicación:
   ```bash
   flutter run
   ```
